# CSS 硬件加速

CSS 硬件加速又叫 GPU 加速。

-   使用硬件加速，可以让`transform`、`opacity`、`filters`这些动画不会引起重绘重排。
-   但是动画其他属性，还是会引起重绘重排，但可以提高性能。

## 使用

常用可以触发硬件加速的 CSS 属性：

-   transform
-   opacity
-   filters
-   will-change

## 优点

-   不会(减少)引起页面的重绘重排，提高性能，加快渲染速度。

## 缺点

-   使用过多的硬件加速，导致内存占用过大，会有性能问题。
-   在 GPU 渲染字体会导致抗锯齿无效。这是因为 GPU 和 CPU 的算法不同。因此如果你不在动画结束的时候关闭硬件加速，会产生字体模糊。
